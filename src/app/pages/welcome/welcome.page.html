GeoHadir memproses data akun, profil karyawan, absensi, lokasi masuk dan pulang, cuti, lembur, persetujuan, penggajian, dan slip gaji untuk kebutuhan sistem HRIS.

Ringkasan Kebijakan Privasi

Data yang diproses

GeoHadir dapat memproses nama, email, nomor telepon, peran, jabatan, departemen, data absensi, lokasi, pengajuan cuti, pengajuan lembur, status persetujuan, dan data penggajian.

Penggunaan lokasi

Data lokasi digunakan saat masuk dan pulang kerja untuk membantu validasi presensi berbasis geolokasi.

Akses berdasarkan peran

Karyawan dapat melihat data pribadinya. Manajer dapat memantau tim dan persetujuan. Admin atau SDM dapat mengelola data sesuai kewenangan.
