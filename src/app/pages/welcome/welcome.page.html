GeoHadir memproses data akun, profil karyawan, absensi, lokasi check-in/check-out, cuti, lembur, approval, payroll, dan slip gaji untuk kebutuhan sistem HRIS.

Ringkasan Kebijakan Privasi

Data yang diproses

GeoHadir dapat memproses nama, email, nomor telepon, role, jabatan, departemen, data absensi, lokasi, pengajuan cuti, pengajuan lembur, status approval, dan data payroll.

Penggunaan lokasi

Data lokasi digunakan saat check-in dan check-out untuk membantu validasi absensi berbasis geolocation.

Akses berdasarkan role

Employee dapat melihat data pribadinya. Manager dapat memantau tim dan approval. Admin atau HR dapat mengelola data sesuai kewenangan.
